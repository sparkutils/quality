ExpressionAggregates

Related Doc: package aggregates

case class ExpressionAggregates(children: Seq[Expression], zero: (DataType) ⇒ Option[Any], addF: (DataType) ⇒ Option[(Expression, Expression) ⇒ Expression]) extends DeclarativeAggregate with Product with Serializable

Represents an aggregation expression built from a filter function, a sum lambda and an evaluate lambda which uses the count of filter hits and the sum as parameters.
