PassThroughCompileEvals 

case class PassThroughCompileEvals(children: Seq[Expression]) extends Expression with PassThrough with CodegenFallback with Product with Serializable

Same as unevaluable but the queryplan runs. This version requires compileEvals = true (rules are independent and will not use Subexpression Elimination at eval time) and as such cannot be used with SubExprEvaluationRuntime
